# Chatovací aplikace v Pythonu (Socket Programming)

Jednoduchá chatovací aplikace využívající Python sockety pro komunikaci mezi více klienty.

## Jak spustit

### 1. Spusťte server
```bash
python server.py
```

### 2. Spusťte klienta (v novém terminálu)
```bash
python client.py
```

Můžete spustit více klientů ve více terminálech – všichni budou komunikovat přes server.

## Funkce
- Více klientů najednou
- Přezdívky uživatelů
- Zprávy se zobrazují v reálném čase
- Oznámení o připojení/odpojení uživatelů
- Příkaz `/quit` pro odpojení

## Požadavky
- Python 3.6+
- Žádné externí knihovny (pouze standardní knihovna)
